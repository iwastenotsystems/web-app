    Controllers Module

  • ¶

    Create a new module called myApp.controllers to act as a container for the application controllers.

    myAppControllers = angular.module('myApp.controllers', []);
  • ¶

    myApp.manifestController

    myApp.manifestController provides the Application’s manifest data which is retreived via the myApp.manifestService object.

    myAppControllers.controller 'myApp.manifestController', [
  '$scope'
  'myApp.manifestService'
  ($scope, manifestService) ->
    $scope.manifest = manifestService.getManifest()
]

    The $routeProvider route path specified an optional named group parameter, url. When available use that value for the Url, otherwise default to README.md.

    Request that the $http object fetch the data from the Url, then set the $scope’s content member to the result’s data when the Promise resolves.
